# 💰 Expense Tracker (MERN Stack)

> A MERN stack expense tracker to manage, visualize, and analyze personal finances efficiently.

---

## ✨ Features
- ➕ Add, edit, and delete expenses  
- 🏷️ Categorize transactions (Food, Travel, Bills, etc.)  
- 📜 View expense history with filters  
- 📊 Interactive charts for expense visualization  
- 🔐 Secure backend with RESTful APIs  
- 📱 Responsive UI for desktop and mobile  

---

## 🛠️ Tech Stack
- **Frontend:** React, React Router, Axios, Chart.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose ORM)  
- **Authentication:** JWT (JSON Web Token)  
- **Styling:** CSS / Tailwind / Material-UI
